# Technology Stack

## Frontend
- **Framework**: Next.js 15+ (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS + Shadcn/ui components
- **State Management**: Zustand + React Query
- **Animation**: Framer Motion
- **Charts**: Recharts

## Backend
- **Runtime**: Node.js 20+
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth
- **API**: Next.js API Routes (App Router)
- **File Storage**: Supabase Storage
- **Caching**: Redis (Upstash)

## AI/ML Services
- **Language Models**: OpenAI GPT-4o, Claude 3.5 Sonnet
- **Speech**: OpenAI Whisper (Speech-to-Text), ElevenLabs (Text-to-Speech)
- **Vector Database**: Pinecone
- **Image Generation**: DALL-E 3
- **Analytics**: PostHog

## Mobile
- **Framework**: React Native 0.74+
- **Navigation**: React Navigation 6+
- **Notifications**: Expo Notifications
- **Offline Storage**: SQLite + WatermelonDB

## Development Tools
- **Monorepo**: Turbo
- **Package Manager**: npm
- **Linting**: ESLint + Prettier
- **Testing**: Jest + React Testing Library + Playwright
- **CI/CD**: GitHub Actions
- **Deployment**: Vercel (Frontend), Railway (Backend services)

## Educational Specific
- **Question Bank**: Custom PostgreSQL schema
- **Performance Analytics**: Custom analytics with PostHog
- **Content Management**: Supabase CMS
- **Video Streaming**: Cloudflare Stream
- **CDN**: Cloudflare

## Third-party Integrations
- **Payment**: Razorpay (Indian market)
- **Email**: Resend
- **SMS**: Twilio
- **Monitoring**: Sentry
- **Analytics**: PostHog + Google Analytics 4
